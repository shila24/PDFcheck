# ファイルPDF判定・変換ツール

ブラウザ内だけでファイルの中身を判定し、PDF は内容を変えず `.pdf` として保存し直し、対応する画像・テキストは PDF として出力する静的 HTML ツールです。すべて端末内で処理し、ファイルを外部へ送信しません。

## 使い方

1. `index.html` をブラウザで開く。
2. ファイルを選択、またはドロップする(複数可)。
3. 「変換対象リスト」で対象の選択(チェック)と順番(↑↓ボタン・ドラッグ)を整える。
4. 出力方法を「画像をまとめる」または「すべて個別」から選び、「PDF化」を押す。結果カードから「PDFをダウンロード」できる。
5. 出力ファイル名は、1件なら `file_<日時>.pdf`、複数件なら `file1_<日時>.pdf`, `file2_<日時>.pdf` のように自動で付きます。

## できること

主な機能は、以下の「拡張子抜けPDFの修復」と「画像をまとめてPDF」です。

- **中身から判定**: 拡張子ではなく `%PDF-` ヘッダーや画像シグネチャで形式を判定。
- **拡張子抜けPDFの修復**: 中身がPDFなら、内容を変えず `.pdf` 付きで保存し直し。
- **画像をまとめてPDF**: 選択した複数画像をリスト順に1つの複数ページPDFへ結合。
- **個別PDF出力**: 「すべて個別」を選ぶと、画像も含めて選択ファイルごとにPDFを出力。
- **テキストをPDF**: UTF-8 / UTF-16 / Shift_JIS / EUC-JP などを判定し、ページ画像としてPDF化。
- **日本語ファイル名**に対応。

## 対応形式

- PDF: `%PDF-` を検出 → 内容を変えず `.pdf` として保存し直し。
- 画像: JPEG / PNG / WebP / GIF / BMP → A4 PDF(「まとめる」では複数画像を1つに結合、「個別」では画像ごとに出力)。
- テキスト: 各種エンコーディングを試行し、ページ画像として PDF に埋め込み。
- SVG / TIFF など: 安定対応外のため非対応理由を表示します。

## できないこと

- Office系(docx / xlsx / pptx)のレイアウト保持変換は、静的HTMLだけでは安定して行えません。
- 既存のPDF同士を1つに結合する機能は提供していません。信頼できるPDF結合には第三者PDFライブラリの同梱が必要で、本ツールは依存ライブラリなし・端末内処理・no-uploadを優先しているためです。

- 本ツールは MIT License のもと、現状の状態のまま提供されます。明示・黙示を問わず一切の保証はなく、ファイル内容についても保証いたしかねます。ご利用は自己責任でお願いします。
- すべての処理は端末内(ブラウザ)で完結し、ファイルや内容を外部へ送信・保存しません。外部リクエスト・外部ライブラリ・解析タグはありません。
- PDFは中身の `%PDF-` ヘッダーで識別し、内容を変えず `.pdf` として保存し直します。PDFの内容検査・修復・無害化は行いません。
- 画像・テキストはページ画像としてPDF化します。生成PDFに検索可能なテキスト層はありません。
- 巨大なファイルはサイズ/メモリ上限を超過して処理できないことがあります。
- 重要な用途では生成結果を必ずご自身で確認してください。機密ファイルの取り扱いは所属組織の規程に従ってください。
